## Repository Overview

This is a personal study/documentation repository containing learning notes written in Traditional Chinese. It covers Go backend development, database operations, and hardware communication protocols.

## Content Structure

| File | Topic |
|------|-------|
| `Gin.md` | Go Gin web framework — context, routing, HTTP codes, struct tags, XORM transactions |
| `XORM.md` | XORM ORM library for Go |
| `DataBaseOp.md` | SQL operations — DELETE vs TRUNCATE vs DROP |
| `Communication Protoco.md` | Hardware protocols — I2C and UART |

## Notes Format

Notes are written in Traditional Chinese and typically cover:
- Concept definitions and explanations
- Code snippets in Go or SQL
- Comparisons between similar technologies/approaches

There are no build, lint, or test commands — this repository contains only documentation.
